
# Daniel Online

> Using a Split Terminal

&nbsp;

## Project Set Up - Frontend
```
cd frontend
npm run serve
```
> To end NPM 'CTRL+C'

&nbsp;

## Project Set Up - Backend
```
cd backend
source env/bin/activate
python app.py
```
> To stop App 'CTRL-C'
> To end ENV 'deactivate'

&nbsp;
